# Requirements

## R1 — State Formation Control
Systems MUST define how internal state is constructed.

## R2 — Dual Admissibility
Systems MUST distinguish between state admissibility and action admissibility.

## R3 — State Lineage
Systems MUST track how state is formed and transformed.

## R4 — Proof Handling
Systems MUST NOT treat proof as absolute correctness.

## R5 — Accountability Origin
Systems MUST define ownership at decision origin.
