# Definitions

## State Formation
The process by which a system constructs its internal representation of reality.

## State Admissibility
Conditions under which a representation is allowed to exist as valid system state.

## Action Admissibility
Conditions under which an action is permitted based on current state.

## Governance Drift
Misalignment between authority, decision production, and accountability.

## Proof Persistence
Stabilization of system behavior through validation mechanisms.
